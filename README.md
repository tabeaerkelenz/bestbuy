# BestBuy OOP Simulation

A Python-based object-oriented simulation of an e-commerce system.

## Purpose
This project was built to practice:
- Object-Oriented Programming (OOP)
- Class design and relationships
- Separation of concerns
- Business logic modeling

## Technologies Used
- Python
- Object-Oriented Design (OOP)

##  Features
- User and product modeling with Python classes  
- Simulated shopping workflow  
- Encapsulation of business logic  
- Clean class structure and method design  

##  How to run
```bash
python main.py
```
